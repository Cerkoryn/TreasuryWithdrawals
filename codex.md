The goal of this repository is to use a template to create and submit 39 individual treasury withdrawal proposals on the Cardano Blockchain.
The 39 specific proposals were already submitted together in a single Budget Info Action that passed the required voting threshold by dReps and the Constitutional Committee
The budget can be found in the working directory as budget.txt
The constitution can be found in the working directory as constitution.md
The list of approved proposals can be found in the working directory as proposals.csv
It is VITAL that the template meets all the constitutional requirements for a treasury withdrawal
It is VITAL that the template perfectly matches the terms established in the approved budget
It is VITAL that each treasury withdrawal proposal meets all the constitutional requirements for a treasury withdrawal
It is VITAL that each treasury withdrawal perfectly matches the terms established in the approved budget
